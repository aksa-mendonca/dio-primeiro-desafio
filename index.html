<h1> ao <h6>

Voltar

São tags usadas para criar cabeçalhos em HTML, onde <h1> é o mais importante e <h6> é o menos importante. Geralmente são usados para organizar e estruturar o conteúdo da página, com <h1> sendo o título principal e <h6> sendo um título menor ou subtítulo.

<p>

Voltar

Esta tag é usada para criar parágrafos de texto em HTML. É uma das mais básicas e frequentemente usadas para estruturar o conteúdo textual de uma página da web.

<mark>

Voltar

A tag <mark> é usada para destacar parte do texto dentro de um parágrafo ou qualquer outro elemento de texto. O texto envolvido por <mark> é geralmente exibido com um fundo amarelo para chamar a atenção do leitor.

<small>

Voltar

Esta tag é usada para definir texto que deve ser exibido em um tamanho menor do que o texto ao redor. É comumente usado para legendas, notas de rodapé ou outros textos secundários.

<i>

Voltar

A tag <i> é usada para enfatizar o texto, geralmente representando um texto em itálico. No entanto, é importante notar que, desde HTML5, a tag <i> é mais semanticamente usada para indicar texto com significado alternativo ou voz de um personagem, em vez de apenas itálico.

<u>

Voltar

A tag <u> é usada para sublinhar o texto em HTML. No entanto, seu uso não é mais recomendado, especialmente porque a sublinhagem geralmente é associada a links na web.

<strong>

Voltar

Esta tag é usada para dar ênfase forte a um texto, geralmente fazendo-o parecer mais importante ou mais destacado. Geralmente é exibido em negrito pelo navegador.

<ol>

Voltar

A tag <ol> é usada para criar listas ordenadas em HTML, onde cada item da lista é precedido por um número ou outro tipo de marcador de ordem.

<ul>

Voltar

Esta tag é usada para criar listas não ordenadas em HTML, onde cada item da lista é precedido por um marcador de ponto.

<li>

Voltar

A tag <li> é usada dentro de <ol> ou <ul> para definir cada item da lista.

<a>

Voltar

Esta é a tag de âncora, usada para criar links em HTML. Ela permite que você vincule uma página a outra ou uma seção a outra dentro da mesma página.

<hr>

Voltar

A tag <hr> é usada para criar uma linha horizontal em HTML, geralmente usada para separar seções de conteúdo.

<sub>

Voltar

Esta tag é usada para definir texto subscrito, onde o texto é exibido ligeiramente abaixo da linha de base normal.

<sup>

Voltar

A tag <sup> é usada para definir texto sobrescrito, onde o texto é exibido ligeiramente acima da linha de base normal.

<blockquote>

Voltar

Esta tag é usada para citar um bloco de texto de outra fonte, geralmente com uma formatação especial para destacá-lo do texto circundante.
